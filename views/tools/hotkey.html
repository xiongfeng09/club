快捷键 描述
F2 高亮错误或警告快速定位
F5 复制文件
F6 移动文件
F9 弹出运行窗口
F11 添加当前文件的书签
F12 显示文件结构
Ctrl + Y 删除行
Ctrl + N 快速查找类
Ctrl + R 替换
Ctrl + O 为当前类选择可覆盖或继承的方法
Ctrl + F 在当前代码中查找
Ctrl + J 提示插入Live Template的代码段
Ctrl + D 向下复制选中内容或者向下复制当前光标所在行内容
Ctrl + W 选中光标所在单词，不断按下W，将不断扩大范围(反向选中: Ctrl + Shift + W)
Ctrl + Q 显示光标所在变量/类名/方法的文档
Ctrl + B 定位光标所在类或者方法的定义
Ctrl + E 显示最近打开的文件(支持搜索过滤)
Ctrl + U 定位当前类的父类或者方法
Ctrl + K 提交项目代码(Commit)
Ctrl + T 更新项目代码
Ctrl + G 定位指定行
Ctrl + H 显示类结构层次图
Ctrl + F1 显示错误
Ctrl + F3 跳转到当前选中词的下一个同名词
Ctrl + F12 显示当前文件大纲/快速导航(支持搜索过滤)
Ctrl + F11 显示添加书签窗口
Ctrl + Tab 切换编辑窗口
Ctrl + Delete 删除当前光标后的单词
Ctrl + Home/End 定位到文件头部/尾部
Ctrl + BackSpace 删除当前光标前的单词
Ctrl + [ 或 ] 移动光标到块的起始和结束位置
Ctrl + 数字 定位标签
Ctrl + 小键盘 +/- 折叠/展开代码
Ctrl + 鼠标单击编辑窗口的文件标题 显示文件路径
Ctrl + 左/右方向键 光标跳到上一个或者下一个单词
Alt + 数字 切换和隐藏面板
Alt + F1 显示文件操作目标窗口

Shift + F6 重构/重新命名

Shift + Enter 光标所在行下空出一行且光标跳下
Shift + 鼠标点击编辑窗口的标题 关闭当前编辑窗口
Shift + 鼠标滚轮 横向滚动面板/编辑窗口
Ctrl + Alt + O 优化导入的类和包
Ctrl + Alt + T 环绕选中的代码提示(可结合Live Template使用)
Ctrl + Alt + B 显示子类或者继承类 / 跳转到实现类的方法
Ctrl + Alt + Enter 光标所在行上空出一行且光标跳上

Ctrl + Alt + 左右方向键 前进或者返回上一个操作位置

Ctrl + Alt + 空格 显示类名或方法的文档或者提示

Ctrl + Alt + N 快速查找文件
Ctrl + Shift + N 快速查找文件
Ctrl + Shift + F 文件查找

Ctrl + Shift + C 复制当前文件所在路径到粘贴板
Ctrl + Shift + J 自动将下一行合并到当前行末尾
Ctrl + Shift + E 显示最近修改的文件
Ctrl + Shift + T 显示重构窗口
Ctrl + Shift + R 搜索指定范围的文件且替换文本
Ctrl + Shift + N 快速查找文件

Ctrl + Shift + F7 高亮显示与所选文本相同的内容

Ctrl + Shift + 数字 快速添加书签
